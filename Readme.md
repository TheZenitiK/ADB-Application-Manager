Данный скрипт предназначен для удаления системных приложений на устройствах под управлением Android с помощью инструмента ADB. Он включает в себя базу системных приложений, удаление которых не нанесет вреда системе (однако, следует помнить, что вы делаете это на свой страх и риск). Скрипт подходит для различных оболочек, таких как MIUI, HyperOS, OriginOS, Realme UI и ColorOS, а также для удаления приложений Google и сторонних приложений, которые разработчики часто предустанавливают на новые устройства. Он подходит для всех устройств Android.

Инструкция по использованию:
Для использования скрипта распакуйте архив в удобное место и запустите файл с расширением `.bat`. Подключите устройство, предварительно включив отладку по USB в настройках для разработчиков. На телефоне также необходимо разрешить компьютеру отладку. 

Когда вы запустите скрипт, нажмите «Enter». Если вы видите зеленую надпись **"Connected"**, значит, устройство подключено и готово к работе. Теперь, используя цифры, выбирайте нужные пункты и приложения для удаления. Если вы случайно удалили нужное приложение, вы можете его восстановить, нажав клавишу **88** и выбрав в меню приложение для восстановления. Если вы не уверены, какое приложение следует восстановить, воспользуйтесь пунктом **99**, который восстановит все удаленные приложения.
